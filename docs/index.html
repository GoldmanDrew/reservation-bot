Reservation Bot

4 inputs → automated OpenTable sniper via GitHub Actions

One-time setup

  1. Sign in with GitHub (top right) — no PAT to paste into Settings
  2. Connect OpenTable below — saves cookies to repo secrets from this page
  3. Add TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID in GitHub Secrets (or save via API later)
  4. Create a snipe with the 4-field form — runs Create Snipe workflow automatically

Connect OpenTable

Paste cookies once here instead of GitHub Settings. Requires admin on the repo.

How to get OpenTable cookies
  1. Log into opentable.com in Chrome
  2. DevTools → Network → refresh
  3. Click any request to www.opentable.com
  4. Copy the full cookie: header value (not a URL)

Create a snipe (4 fields)

What happens automatically

  • Detects OpenTable vs Resy
  • Resolves restaurant ID (rid)
  • Calculates drop time (e.g. 30 days @ 9am ET for San Sabino)
  • Writes config/snipes.yaml and commits
  • Telegram confirmation when snipe is armed
  • Every 5 min: polls at drop time, sends booking link when slot found

Manual fallback

Actions → Create Snipe → Run workflow
