Reservation Bot

Free GitHub Actions sniper for OpenTable & Resy

How it works

  1. Add secrets to GitHub Secrets:
    • OpenTable: OPENTABLE_COOKIES (required for OT)
    • Resy: RESY_EMAIL + RESY_PASSWORD (optional)
    • Alerts: TELEGRAM_BOT_TOKEN + TELEGRAM_CHAT_ID (recommended) or WEBHOOK_URL (Discord)
  2. Generate a snipe below → paste into config/snipes.yaml
  3. Push to GitHub — Actions runs every 5 minutes
  4. OpenTable: click the booking link in Actions summary when a slot is found

OpenTable cookies (one-time setup)

  1. Log into opentable.com in Chrome
  2. F12 → Network → refresh the page
  3. Click any request to opentable.com
  4. Copy the full Cookie header value
  5. Paste as OPENTABLE_COOKIES in GitHub Secrets
  6. Optional: copy x-csrf-token → OPENTABLE_CSRF_TOKEN

Cookies expire every few days — re-paste when snipes fail with auth errors.
