Reservation Bot

Free GitHub Actions sniper for Resy — inspired by 0xLoris

How it works

  1. Add RESY_EMAIL and RESY_PASSWORD to GitHub Secrets
  2. Generate a snipe below and paste it into config/snipes.yaml
  3. Push to GitHub — Actions runs every 5 minutes automatically
  4. Check Actions tab for results
